# Skill: Turn Retrospective

## Goal
Compress a completed turn into one useful lesson.

## Use This Skill When
- A turn just finished and you can judge whether it was elegant or clumsy.
- You want a terse, reusable lesson rather than a long postmortem.

## Do Not Use This Skill When
- The turn was trivial.
- There is not yet enough evidence to judge what the better path was.

## Template
- **What happened**
- **What made it hard**
- **What would have been better**
- **What should be remembered next time**

## Output
- One compact retrospective suitable for storage in a session ledger.
