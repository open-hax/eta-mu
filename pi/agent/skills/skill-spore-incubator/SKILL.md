# Skill: Skill Spore Incubator

## Goal
Capture reusable capability early without prematurely turning it into a full live skill.

## Use This Skill When
- A hard turn suggests a reusable skill.
- You want a draft artifact with a name, scope, and promotion gate.

## Do Not Use This Skill When
- The problem was obviously one-off.
- You already know you want a finished skill right now.

## Spore fields
- Name
- One-sentence description
- Reuse scope: `turn`, `session`, or `multi-session`
- Source lesson
- Better path next time
- Promotion gate

## Promotion rule
- A spore becomes a full skill after recurrence, explicit request, or strong evidence of generality.

## Output
- A draft skill spore artifact that can later be promoted with `skill-authoring`.
